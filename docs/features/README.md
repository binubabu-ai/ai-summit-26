## Status Legend
- **Draft**: Initial specification being written
- **Planned**: Specification complete, ready for implementation
- **In Progress**: Currently being implemented
- **In Review**: Implementation complete, in code review
- **Done**: Merged and deployed

## Creating a New Feature
Use the `/feature-new` command or copy `_TEMPLATE.md` to create a new feature specification.
